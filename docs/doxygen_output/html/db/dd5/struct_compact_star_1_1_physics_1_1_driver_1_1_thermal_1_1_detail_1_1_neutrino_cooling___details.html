Detailed Description

Bundle of derived neutrino-cooling quantities for physics/diagnostics.

All fields are computed outputs based on the driver options, state vector, and driver context. If a required piece of infrastructure is missing (e.g. geometry/structure tables for volume integration), ok=false is set and message provides a human-readable reason.

Units (policy)

  • Temperatures: [K]
  • Luminosities at infinity: [erg/s]
  • Heat capacity: [erg/K]
  • dT/dt: [K/s]
  • d/dt ln(Tinf/Tref): [1/s]
