Detailed Description

Thermal utilities: cooling/heating helpers.

Contains static thermal functions (e.g., neutrino luminosities) and interfaces used by thermal drivers.

Direct Urca neutrino emissivity (Q_\nu^{\text{DUrca}}(r)).

Returns the prefactor-only emissivity in ([ \mathrm{erg}\ \mathrm{cm}^{-3}\ \mathrm{s}^{-1} ]) so that callers can scale with ( (T_\infty/10^9\,\mathrm{K})^6 ) when desired.

Redshifted photon luminosity (L_\infty) from the surface.

Uses Stefan–Boltzmann law with gravitational redshift. Assumes thin blanket (R_blanket ≈ R) so that (T_\infty = z_\text{surf}\, T_\text{surf}).

Parameters
viewNon-owning structural profile view (needs R and z_surf).
stateThermal state (uses T_surf).
Returns
Redshifted luminosity (L_\infty) in [erg s(^{-1})].
Note
If we prefer to use a (T_\infty)-based fit (e.g., (T_\infty^{2.42})), we should implement a sibling function instead of overloading this one.
