Detailed Description

Driver for surface photon cooling contribution to dT̃/dt.

Converts (L_{\gamma,\infty} = 4\pi R^2 \sigma T_{\text{surf}}^4 z_\text{surf}^2) (or an envelope model provided via StarContext) into the redshifted thermal RHS term −L_γ/C_eff. Envelope physics (T_core↔T_surf relations) should be provided by the context/microphysics layer.
