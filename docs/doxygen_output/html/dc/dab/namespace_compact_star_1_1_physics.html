Detailed Description

High-level physical modeling layer for compact stars.

Provides thermal, chemical, spin, and BNV physics modules built on top of Core structures (StarProfile, NStar, EOS). This layer contains:

  • Evolution drivers
  • Thermal / spin / chem utilities
  • State blocks
  • Evolution system components

This namespace serves as the umbrella for all physics submodules in: CompactStar/Physics/
