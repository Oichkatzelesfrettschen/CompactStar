Detailed Description

Class representing a table of equation of state (EOS) data.

The EOSTable class stores the energy density, pressure, and baryon number density values for a given EOS. It also provides methods to set labels, add extra labels, and print the table.

void CompactStar::Core::EOSTable::Print ( const int  n_lines = 5) const
inline

Printer for the table.

Prints the table to the standard output.

Prints the EOS table to the standard output.

  • By default, prints only the header and the first few lines (5).
  • If n_lines is specified and positive, prints up to that many rows.
  • If n_lines exceeds the table size, prints the entire table.
  • Negative values for n_lines (e.g. -1) force printing all rows.
    Parameters
    n_linesOptional number of data rows to print (default = 5).

Prints a compact summary of the EOS table.

Shows:

  • Number of data rows.
  • Column labels.
  • Minimum and maximum values of ε, P, and ρ.
  • Extra species columns, if present. Intended for quick sanity checks before running the TOV solver.
