Detailed Description

Neutron-star container for TOV solutions, rotation, and export.

Transitional “phase 2” class:

  • new path: TOV → StarProfile (preferred)
  • old path: TOV → ds (kept, but marked for replacement)

int NStar::SolveTOV_Profile ( const Zaki::String::Directory &  eos_file,

Solve TOV for a single star and populate this NStar's StarProfile.

Parameters
eos_fileFull path (directory + filename) of the EOS table, e.g. eos_root + eos_name + "/" + eos_name + ".eos".
target_M_solarTarget gravitational mass in units of M_sun.
rel_out_dirOptional subdirectory (relative to this NStar's working directory) where the TOV solver can dump any sequence / debug output if it wants.
Returns
Number of radial points in the final profile on success, or 0 on failure.

This is a high–level one-shot constructor:

  • builds an internal TOVSolver,
  • imports the EOS from eos_file,
  • calls TOVSolver::SolveToProfile(...),
  • then calls BuildFromTOV(...) to populate prof_ and all derived quantities (B, I, z_surf, etc.).
