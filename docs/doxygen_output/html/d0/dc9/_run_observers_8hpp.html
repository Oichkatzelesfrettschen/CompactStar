Detailed Description

Standard observer option presets and factories for Evolution runs.

This module centralizes run-level observer policy so debug mains stay small:

  • DiagnosticsObserver (JSONL packets + optional catalog)
  • TimeSeriesObserver (CSV/TSV table; optionally schema-driven via catalog)

This is intentionally separate from Observers/ implementation: Observers are generic components; this file encodes “how we like to run them”.
