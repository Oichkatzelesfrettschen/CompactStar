40// Updated on Mar 15, 2023
41// This class is made specifically for analyzing the
42// B --> Chi Gamma
43// decays in neutron stars. It is very similar in
44// structure to "Decay_Analysis" class which was made
45// earlier to evaluate the same decay rate.
46// "Decay_Analysis" decay formula in NS is wrong.
47// "BNV_B_Chi_Photon" fixes the mistakes in
48// "Decay_Analysis". We are keeping the
49// "Decay_Analysis" files for now, but might delete it
50// in the future.
