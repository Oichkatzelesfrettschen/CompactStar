bool CompactStar::Physics::Driver::Thermal::Boundary::TbDefinition::assume_isothermal_redshifted = true

If true, Tb is computed from isothermal redshifted T_inf:

Tb = T_inf * exp(-ν(r_b))

Otherwise, caller may supply Tb directly or compute via another model.
