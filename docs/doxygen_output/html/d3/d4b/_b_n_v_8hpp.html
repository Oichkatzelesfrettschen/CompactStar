Detailed Description

Baryon-number-violation utilities built on profiles + spin state.

Provides helper routines to estimate bounds on the total BNV rate from pulsar spin-down, and to populate the BNVState block with corresponding diagnostics (e.g. Î“_BNV limits).

Spin-down energy budget bound on the total BNV rate.

Computes an upper limit on by requiring the BNV heating power not to exceed the observed spin-down power (model assumptions apply).

This is a pure utility function and does not modify any state blocks.

Compute the spin-down bound and store it in a BNVState block.

Convenience wrapper around SpinDownLimit(...) that writes the resulting bound into the BNV evolution/diagnostic state. This keeps BNV-related quantities available to drivers and observers without recomputing them.

By convention, the spin-down limit is stored in bnv.SpinDownLimit() matching the accessor provided by Physics::State::BNVState.
