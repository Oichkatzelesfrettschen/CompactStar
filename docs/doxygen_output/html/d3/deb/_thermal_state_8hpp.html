Detailed Description

Thermal evolution state block for a compact star.

This class provides the dynamical thermal degrees of freedom used by the evolution system (typically the redshifted internal temperature T∞, or multiple temperature components if needed), as well as optional auxiliary temperatures used by microphysics or cooling models.

Directory-based namespace: CompactStar::Physics::State

Design

  • The evolution DOFs live in a contiguous vector (values_).
  • A typical minimal configuration has just one DOF = T∞.
  • Subclasses or callers may configure multiple components by calling Resize().

Auxiliary variables

The local-frame temperatures (T_core, T_blanket, T_surf) are not part of the ODE vector. They are cached values used by cooling models, envelopes, emissivity computations, and diagnostics.
