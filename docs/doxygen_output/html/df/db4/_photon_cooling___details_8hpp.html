Detailed Description

Shared computations for PhotonCooling (physics + diagnostics).

This file centralizes the computation of:

  • surface temperature mapping (currently trivial / placeholder),
  • A_inf and A_eff_inf,
  • luminosity at infinity,
  • physical cooling rate dTinf/dt [K/s],
  • ODE RHS contribution d/dt ln(Tinf/Tref) [1/s].

The goal is to avoid the “diagnostics drift” problem: if you change the physics implementation, diagnostics automatically match, because they call the same helpers.
