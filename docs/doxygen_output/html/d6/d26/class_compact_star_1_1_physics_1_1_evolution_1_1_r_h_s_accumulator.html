Detailed Description

Write-only accumulator for dY/dt components, keyed by StateTag.

Internally, this class maintains a small array of "blocks", one per StateTag. Each block is a vector<double> of length equal to the number of DOFs for that state. Drivers contribute via AddTo(tag, i, value), and the evolution system reads the final RHS via Block(tag).
