MicroBNVInt::BNV_Chi::Rate_Eps MicroBNVInt::BNV_Chi::GetRate_Eps ( const double &  m_chi,
const Baryon &  B 
)

Integrate the B → χ decay rate over the star and return (i) the total baryon-number-loss rate per baryon [yr⁻¹] (ii) the corresponding limit on ε for this channel.

Returns the rate and limit on eps from B -> chi decay rate integrated over the radius and in units of [ s^-1 ] & MeV

This does:

  1. grab r, M(r), ν(r) from the pulsar’s StarProfile
  2. get the local decay rate vs radius from Rate_vs_R(m_chi, B)
  3. apply GR volume element 4π r² (1 - 2M/r)^(-1/2) and redshift e^{ν}
  4. convert km³·fm⁻³ → dimensionless via 1e54
  5. integrate over r
  6. convert s⁻¹ → yr⁻¹
  7. divide by total baryon number B_tot from the sequence point
  8. rescale ε: ε_limit = ε₀ √( Γ_sd / Γ_model )
