Detailed Description

Evolution engine for time-dependent neutron-star physics.

Contains:

  • EvolutionSystem (RHS functor)
  • StateVector registry
  • RHSAccumulator
  • GeometryCache
  • Integrator bindings
  • Observers, logging, checkpoints

This namespace represents the core of the time-evolution subsystem.

double *  y

Pack all active State blocks into the flat ODE vector y[].

For each active tag in layout, this:

  1. obtains the corresponding State& from state,
  2. determines its block size from layout,
  3. calls State::PackTo() to write into y[offset..offset+size).
Parameters
stateComposite view over all registered State blocks.
layoutLayout describing offsets/sizes for each tag in y[].
yOutput pointer to the first element of the flat ODE vector. Must have capacity at least layout.TotalSize().

const StateLayout &  layout,
double *  dydt 
)

Scatter contributions from RHSAccumulator into dydt[].

After drivers have written dY/dt contributions into rhs (one block per StateTag), this function copies those blocks into the flat RHS vector dydt[] using the offsets/sizes stored in layout.

Only tags that are active in layout and configured in rhs are copied. For safety, a mismatch between block sizes will throw.

Parameters
rhsRHSAccumulator containing per-tag dY/dt blocks.
layoutLayout describing offsets/sizes for each tag in dydt[].
dydtOutput pointer to the first element of the flat RHS vector. Must have capacity at least layout.TotalSize().

Unpack the flat ODE vector y[] back into the State blocks.

For each active tag in layout, this:

  1. obtains the corresponding State& from state,
  2. determines its block size from layout,
  3. calls State::UnpackFrom() on the slice y[offset..offset+size).
