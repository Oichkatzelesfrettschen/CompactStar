Detailed Description

Structural / metric / composition data for a compact-star configuration.

One instance = one star.

It contains:

  • full radial data as a DataSet;
  • sequence metadata (M, R, central values) via SeqPoint;
  • explicit indices for all “core” columns (r, m, ν′, p, ε, ρ, ν, λ);
  • a typed enum class Column so new code can use enum and old code can use ints;
  • per-species density columns, with names and indices.

Scoped identifiers for common profile columns.

We match the legacy layout: 0: Radius 1: Mass 2: MetricNuPrime 3: Pressure 4: EnergyDensity 5: BaryonDensity 6: MetricNu 7: MetricLambda

MetricNuPrime

Pressure 

p(r)

EnergyDensity 

ε(r)

BaryonDensity

MetricNu 

ν(r) such that g_tt = -e^{2ν}

MetricLambda 

λ(r) such that g_rr = e^{2λ}
