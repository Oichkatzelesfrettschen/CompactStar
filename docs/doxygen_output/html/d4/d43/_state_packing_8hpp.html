Detailed Description

Helpers for packing/unpacking State blocks to/from flat ODE arrays.

These free functions provide the glue between:

  • the tagged State blocks stored in StateVector,
  • the layout information in StateLayout, and
  • the flat y[] / dydt[] arrays used by GSL.

They rely on the virtual PackTo / UnpackFrom methods provided by the abstract State base class and implemented by each concrete State block (SpinState, ThermalState, ChemState, BNVState, ...).
