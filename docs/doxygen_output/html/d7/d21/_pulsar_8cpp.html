Detailed Description

Source for the lightweight, modular Pulsar class.

This implementation corresponds to the new Pulsar interface in CompactStar/Core/Pulsar.hpp, i.e. the one that:

  • owns a StarProfile (prof_)
  • exposes a StarProfileView (view_)
  • stores spin/thermal/BNV state in tiny PODs
  • delegates all actual physics to the headers in CompactStar/Physics/

The goal is to keep this file SHORT. Anything that starts looking like "derive emissivity as a function of radius" goes into Physics/Thermal.cpp (or the header-only version of it).
