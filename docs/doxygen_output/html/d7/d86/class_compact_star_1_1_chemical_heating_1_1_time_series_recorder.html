Detailed Description

Writes samples to a Zaki::Vector::DataSet as columns.

Column layout (created on first use): t[s], Tinf[K], Ts[K], Lnu_inf[erg/s], Lgamma_inf[erg/s], H_inf[erg/s], C_tot[erg/K], eta_0, eta_1, ...
