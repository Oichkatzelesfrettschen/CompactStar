Detailed Description

Immutable, per-star adapter exposing cached geometry and composition.

Constructed from an existing CompactStar::Core::StarProfile. Caches non-owning pointers to frequently accessed columns (r, m, nu, lambda, nb). Validates consistency (row counts) up front to fail fast before evolution.
