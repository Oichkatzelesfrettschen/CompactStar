Detailed Description

Accumulates the thermal RHS from chemical heating (η⋅Γ terms).

Depends on: Chem, Thermal Updates: Thermal

The microphysics for Γ_i should be accessed via the StarContext and/or Microphysics modules (rates/emissivities). This driver only performs the bookkeeping to add its contribution to the thermal component of dY/dt.
