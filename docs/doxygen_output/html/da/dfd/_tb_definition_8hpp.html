Detailed Description

Definition and helpers for the base-of-envelope temperature Tb.

Envelope fits are typically expressed as Ts(Tb, g14, xi, ...), where Tb is the local temperature at the base of the heat blanketing envelope, often chosen at a fiducial density:

ρ_b ≈ 1e10 g/cm^3

The interior is frequently assumed isothermal in the redshifted sense:

T(r) = T_inf * exp(-ν(r))

where T_inf is constant through the isothermal region.

This module encodes:

  • A canonical "base density" choice.
  • A strategy for picking the radius/index of the base-of-envelope.
  • A helper that computes Tb from T_inf and the metric at the base.

Important: this module is policy, not EOS microphysics. If we later implement crust/envelope conduction explicitly, this can be bypassed.

Find the index of the base-of-envelope radius (r_b) in the star grid.

Strategy:

  • Use the StarContext profile to find the first radius where ρ(r) <= ρ_b (i.e., moving outward from core to crust/envelope).

Requirements:

  • StarContext::Profile() exposes density column ρ(r) in g/cm^3 and radius grid R (km).
Exceptions
std::runtime_errorif density profile is missing or the threshold cannot be bracketed.
