Detailed Description

Abstract interface for all evolution drivers.

Drivers contribute terms to the RHS dY/dt for one or more state components. They may read other components (dependencies) to compute their update.

Add this driver’s contribution to the global RHS dY/dt.

Implementations must accumulate into dYdt (i.e., add), and must not assume any specific call order beyond what the Evolution graph resolves.

Parameters
tCurrent time (model units; typically seconds).
YRead-only composite state vector (Spin/Thermal/Chem/…).
dYdtWrite-only accumulator for RHS components (add to it).
ctxRead-only star/context data (geometry, EOS, microphysics).
