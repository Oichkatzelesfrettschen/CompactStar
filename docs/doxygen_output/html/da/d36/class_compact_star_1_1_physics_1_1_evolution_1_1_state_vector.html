Detailed Description

Composite view over the current ODE state (tag → State*).

This class does not store any actual data; it only keeps pointers to State-derived objects, indexed by StateTag. It is intended to be constructed and owned by the evolution System, then passed by const reference into driver calls.

Ownership:

  • StateVector is non-owning; the caller must ensure the underlying State objects outlive the StateVector.
