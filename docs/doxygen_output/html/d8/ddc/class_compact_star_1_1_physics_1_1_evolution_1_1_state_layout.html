Detailed Description

Records offsets and sizes of each StateTag block in the flat ODE vector.

The layout is configured once (typically before the first call to the integrator) and is then treated as immutable during a given evolution run.

It does not store any data itself; it only knows how the y[] / dydt[] arrays are partitioned into tagged blocks.

Configure layout from a StateVector and an ordered list of tags.

The tags in tags define the concatenation order in the flat ODE vector y[]. For each tag:

  • StateVector::Get(tag).Size() is queried to determine block size;
  • a contiguous block [offset, offset+size) is assigned in y[];
  • the internal Block entry is marked active.

Blocks for tags not listed in tags remain inactive.

tagsOrdered list of StateTag values to include in y[].
