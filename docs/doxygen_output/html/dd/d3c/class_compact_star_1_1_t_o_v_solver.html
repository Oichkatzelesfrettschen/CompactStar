Print (at most) max_rows rows of the EOS table.

Parameters
max_rowsMaximum number of data rows (after the header) to print. If max_rows == 0 the function only prints the header and a short footer.

This is useful for large CompOSE tables: you can quickly verify that the file was read correctly, that labels were parsed, and that the first few rows are monotonic in pressure, without flooding the terminal.

◆ SetCentralEDensFloorFactor()

void TOVSolver::SetCentralEDensFloorFactor ( double  f)

Set the floor multiplier for central energy density.

Parameters
fFactor by which the minimum EOS energy density is multiplied to form the lowest allowed central energy density in Solve(...).

If the EOS minimum energy density is , then the solver will clamp user inputs to at least .

This is to avoid trying to build stars whose central energy density is already too close to the surface value of the EOS, which typically makes the TOV integration stop immediately.

Note
A typical value is 10.0.

◆ central_eps_floor_factor

double CompactStar::TOVSolver::central_eps_floor_factor = 10.0
protected

Multiplier applied to the EOS minimum energy density to build a lower bound for central energy density.

If the user provides an axis with values below central_eps_floor_factor * eos_tab.eps.front(), the solver will clamp to that floor.

Default: 10.0.
