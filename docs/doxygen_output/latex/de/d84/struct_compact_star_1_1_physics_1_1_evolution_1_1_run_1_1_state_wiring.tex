\doxysubsection{Detailed Description}
Bundle of Evolution state wiring objects commonly needed by a main(). 

A debug main typically needs all of these\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_state_vector}{State\+Vector}}\+: block registry and view into concrete state blocks
\item \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_state_layout}{State\+Layout}}\+: packing order and total dimension
\item \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_r_h_s_accumulator}{RHSAccumulator}}\+: per-\/tag RHS accumulation buffers
\end{DoxyItemize}

This struct keeps them together for ergonomic use.

Ownership\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_state_vector}{State\+Vector}} stores references/copies depending on your implementation; this struct only stores the \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_state_vector}{State\+Vector}} instance itself.
\item The concrete state blocks (Thermal\+State, Spin\+State, etc.) are owned by your main. 
\end{DoxyItemize}
