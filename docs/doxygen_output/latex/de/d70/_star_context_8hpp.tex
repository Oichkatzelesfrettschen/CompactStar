\doxysubsection{Detailed Description}
Read-\/only bridge to Core structures (NStar, EOS, Rotation\+Solver) for the evolution module. 

Star\+Context centralizes access to geometry, composition, and metric quantities derived from a precomputed neutron-\/star profile. It {\itshape does not} run TOV or rotation; it only exposes cached, interpolation-\/friendly views needed during RHS evaluations.
