\doxysubsection{Detailed Description}
Interface for evolution drivers that contribute to the RHS d\+Y/dt. 

A driver\+:
\begin{DoxyItemize}
\item declares which state blocks it reads (Depends\+On)
\item declares which state blocks it updates (Updates)
\item accumulates its contribution to the global RHS via Accumulate\+RHS(...)
\end{DoxyItemize}

Examples\+:
\begin{DoxyItemize}
\item Driver\+::\+Spin\+::\+Magnetic\+Dipole (Spin → Spin)
\item Driver\+::\+Chem\+::\+Rotochemical (Spin → Chem)
\item Driver\+::\+Thermal\+::\+Neutrino\+Cooling (Thermal → Thermal)
\item Driver\+::\+Thermal\+::\+Heating\+From\+Chem (Chem, Thermal → Thermal)
\end{DoxyItemize}

Implementations should be pure functions of (t, Y, ctx, options) and must accumulate (add) into d\+Y/dt, never overwrite it.
