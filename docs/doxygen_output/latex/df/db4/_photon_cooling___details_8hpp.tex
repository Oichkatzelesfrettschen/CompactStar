\doxysubsection{Detailed Description}
Shared computations for Photon\+Cooling (physics + diagnostics). 

This file centralizes the computation of\+:
\begin{DoxyItemize}
\item surface temperature mapping (currently trivial / placeholder),
\item A\+\_\+inf and A\+\_\+eff\+\_\+inf,
\item luminosity at infinity,
\item physical cooling rate d\+Tinf/dt \mbox{[}K/s\mbox{]},
\item ODE RHS contribution d/dt ln(Tinf/\+Tref) \mbox{[}1/s\mbox{]}.
\end{DoxyItemize}

The goal is to avoid the “diagnostics drift” problem\+: if you change the physics implementation, diagnostics automatically match, because they call the same helpers.
