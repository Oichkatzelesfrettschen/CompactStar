\doxysubsection{Detailed Description}
Abstract base for all ODE “states” (chemical, thermal, spin, …). 

Concrete subclasses must\+:
\begin{DoxyItemize}
\item provide a {\bfseries{contiguous}} storage exposed via \mbox{\hyperlink{class_compact_star_1_1_state_a1908fba8001d6684319674355803f0dc}{Data()}}/\+Size(),
\item implement fixed-\/size setup via Resize(\+N) (where N is usually the radial grid size),
\item implement Na\+N/range checks in \mbox{\hyperlink{class_compact_star_1_1_state_a65c521d97c79f2ec86ce6cf240e8a847}{Sanity\+Check()}}.
\end{DoxyItemize}

Optional (but recommended)\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{class_compact_star_1_1_state_a364c3640e904726fb1a0be3b4df11613}{Export\+Columns()}}\+: append labeled columns to a Data\+Set for debug/\+IO. 
\end{DoxyItemize}
