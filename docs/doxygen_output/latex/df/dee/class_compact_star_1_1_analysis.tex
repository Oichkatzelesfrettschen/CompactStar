\doxysubsection{Detailed Description}
Abstract base class for implementing neutron star (NS) analysis routines. 

The {\ttfamily \mbox{\hyperlink{class_compact_star_1_1_analysis}{Analysis}}} class is a virtual interface meant to be extended by specific analysis strategies. It provides hooks for analyzing both pure neutron stars ({\ttfamily \mbox{\hyperlink{class_compact_star_1_1_n_star}{NStar}}}) and dark-\/visible mixed stars ({\ttfamily \mbox{\hyperlink{class_compact_star_1_1_mixed_star}{Mixed\+Star}}}) as part of TOV sequence scans.

Results can be exported using the {\ttfamily \mbox{\hyperlink{class_compact_star_1_1_analysis_aa24c5cec2864ded99241ddba1431b217}{Export()}}} method. A label is also associated with the analysis for easy identification.
