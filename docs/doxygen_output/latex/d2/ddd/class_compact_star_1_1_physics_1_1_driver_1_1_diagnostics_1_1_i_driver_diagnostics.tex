\doxysubsection{Detailed Description}
Interface that a driver can implement to expose diagnostics. 

This is intentionally orthogonal to \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_i_driver}{IDriver}}\textquotesingle{}s runtime API\+:
\begin{DoxyItemize}
\item If a driver doesn\textquotesingle{}t implement it, nothing changes.
\item If it does, an observer can call \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_driver_1_1_diagnostics_1_1_i_driver_diagnostics_a282ccc724f51a04081b59ca8e6eea0bb}{Diagnose\+Snapshot()}} and \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_driver_1_1_diagnostics_1_1_i_driver_diagnostics_a11b182179ee903feb396c131ca0404d8}{Unit\+Contract()}}. 
\end{DoxyItemize}
