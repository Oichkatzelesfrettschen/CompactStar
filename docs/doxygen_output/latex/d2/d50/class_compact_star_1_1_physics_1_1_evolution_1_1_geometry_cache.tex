\doxysubsection{Detailed Description}
Geometry-\/only cached columns for repeated integrals. 

This class is intended to be\+:
\begin{DoxyItemize}
\item Cheap to copy/move (columns store vectors internally),
\item Purely geometric (no microphysics),
\item Data\+Column-\/native (supports algebra and Mathematica-\/like indexing). 
\end{DoxyItemize}
