\doxysubsection{Detailed Description}
Class representing a table of equation of state (EOS) data. 

The \mbox{\hyperlink{struct_compact_star_1_1_core_1_1_e_o_s_table}{EOSTable}} class stores the energy density, pressure, and baryon number density values for a given EOS. It also provides methods to set labels, add extra labels, and print the table.
