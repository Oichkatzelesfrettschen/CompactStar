\doxysubsection{Detailed Description}
Runtime container for a single diagnostics “snapshot”. 

A Diagnostic\+Packet is a light-\/weight structure that stores\+:
\begin{DoxyItemize}
\item metadata about where/when the snapshot was created (time, step, driver name),
\item a stable list of scalar values (key -\/\texorpdfstring{$>$}{>} number + optional unit + description),
\item optional text blocks (contracts, warnings, notes).
\end{DoxyItemize}

It is intended for\+:
\begin{DoxyItemize}
\item terminal-\/friendly structured printing (via custom renderers),
\item JSON/\+JSONL export for tooling and regression tests.
\end{DoxyItemize}

Notes on design\+:
\begin{DoxyItemize}
\item This module intentionally stores {\itshape only doubles} as numeric payloads.
\item Unit strings are metadata only (no unit arithmetic); they support auditing.
\item Storage uses an ordered map (std\+::map) for deterministic output ordering. 
\end{DoxyItemize}
