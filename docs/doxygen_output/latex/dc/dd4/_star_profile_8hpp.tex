\doxysubsection{Detailed Description}
Unified structural / metric / composition profile for compact stars. 

This is the “fully populated” phase-\/2 version. It mirrors the column layout you already used in {\ttfamily NStar}\+:

0\+: r (radius) 1\+: m (enclosed mass) 2\+: nu\textquotesingle{} (dν/dr) 3\+: p (pressure) 4\+: eps (energy density) 5\+: rho (total baryon number density) 6\+: nu (metric exponent)

plus optional\+: 7\+: lambda (the other metric exponent / g\+\_\+rr)

and then {\bfseries{per-\/species}} density columns after that.

The idea is\+: if the TOV solver wrote it, this struct should be able to carry it.
