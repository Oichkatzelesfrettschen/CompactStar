\hypertarget{_envelope_potekhin2003_8cpp_autotoc_md13}{}\doxysubsubsection{Implementation note}\label{_envelope_potekhin2003_8cpp_autotoc_md13}
The true Potekhin et al. 2003 fits are often provided as piecewise/log-\/polynomial expressions for Ts(Tb, g14, composition/light-\/element column).

For now, these classes forward to the already-\/available Potekhin1997 implementations to keep the code path functional while preserving a stable \char`\"{}2003\char`\"{} API.

Replace the forwarding with the real 2003 formulas when you are ready; no driver code should need to change.
