\doxysubsection{Detailed Description}
Class representing a sequence of TOV solution points. 

The \mbox{\hyperlink{class_compact_star_1_1_sequence}{Sequence}} class stores a vector of \mbox{\hyperlink{class_compact_star_1_1_seq_point}{Seq\+Point}} objects, which represent individual TOV solution points. It provides methods to add points, export the sequence to a file, combine sequences, and clear the sequence.
