\doxysubsection{Detailed Description}
Immutable, per-\/star adapter exposing cached geometry and composition. 

Constructed from an existing {\ttfamily \mbox{\hyperlink{struct_compact_star_1_1_core_1_1_star_profile}{Compact\+Star\+::\+Core\+::\+Star\+Profile}}}. Caches non-\/owning pointers to frequently accessed columns (r, m, nu, lambda, nb). Validates consistency (row counts) up front to fail fast before evolution.
