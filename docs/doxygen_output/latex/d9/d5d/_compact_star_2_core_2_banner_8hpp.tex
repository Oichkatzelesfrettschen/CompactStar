\doxysubsection{Detailed Description}
Process-\/wide program banner utilities. 

Provides a one-\/shot banner display that runs automatically before {\ttfamily main()} via a TU-\/scope static in Banner.\+cpp. We can also call Show\+Banner\+Once() manually if desired (e.\+g., in tests).
