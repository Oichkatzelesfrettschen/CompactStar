\doxysubsection{Detailed Description}
Accumulates the thermal RHS from chemical heating (η⋅Γ terms). 

{\bfseries{Depends on\+:}} Chem, Thermal {\bfseries{Updates\+:}} Thermal

The microphysics for Γ\+\_\+i should be accessed via the Star\+Context and/or Microphysics modules (rates/emissivities). This driver only performs the bookkeeping to add its contribution to the thermal component of d\+Y/dt.
