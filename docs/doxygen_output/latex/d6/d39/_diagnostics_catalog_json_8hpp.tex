\doxysubsection{Detailed Description}
Minimal JSON writer for diagnostics catalog. 

Writes a stable “catalog schema” describing all producers and scalar descriptors.

We intentionally do not use a full JSON library\+:
\begin{DoxyItemize}
\item avoids bringing in dependencies,
\item keeps output deterministic and easy to diff,
\item sufficient for tooling/regression tests.
\end{DoxyItemize}

Determinism\+:
\begin{DoxyItemize}
\item Producers iterate in key order due to std\+::map in Diagnostic\+Catalog.
\item Scalars preserve insertion order from each Producer\+Catalog.
\item Contract lines preserve insertion order. 
\end{DoxyItemize}
