\doxysubsection{Detailed Description}
Write-\/only accumulator for d\+Y/dt components, keyed by State\+Tag. 

Internally, this class maintains a small array of \char`\"{}blocks\char`\"{}, one per State\+Tag. Each block is a vector$<$double$>$ of length equal to the number of DOFs for that state. Drivers contribute via Add\+To(tag, i, value), and the evolution system reads the final RHS via Block(tag).
