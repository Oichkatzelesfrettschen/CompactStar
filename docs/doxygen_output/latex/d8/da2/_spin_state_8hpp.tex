\doxysubsection{Detailed Description}
Spin/kinematic state block + observables for a compact star. 

This class serves two closely related roles\+:
\begin{DoxyItemize}
\item provides a {\bfseries{dynamic state block}} for spin evolution (e.\+g. Ω), exposed to the ODE integrator via the State base interface;
\item stores {\bfseries{observational spin/kinematic quantities}} (P, Pdot, μ, d) as \mbox{\hyperlink{class_zaki_1_1_math_1_1_quantity}{Zaki\+::\+Math\+::\+Quantity}} for use in BNV bounds, torque models, etc.
\end{DoxyItemize}
