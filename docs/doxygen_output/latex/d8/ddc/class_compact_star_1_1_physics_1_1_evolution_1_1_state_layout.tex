\doxysubsection{Detailed Description}
Records offsets and sizes of each State\+Tag block in the flat ODE vector. 

The layout is configured once (typically before the first call to the integrator) and is then treated as immutable during a given evolution run.

It does not store any data itself; it only knows how the y\mbox{[}\mbox{]} / dydt\mbox{[}\mbox{]} arrays are partitioned into tagged blocks.

Configure layout from a \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_state_vector}{State\+Vector}} and an ordered list of tags. 

The tags in {\ttfamily tags} define the concatenation order in the flat ODE vector y\mbox{[}\mbox{]}. For each tag\+:


\begin{DoxyItemize}
\item State\+Vector\+::\+Get(tag).Size() is queried to determine block size;
\item a contiguous block \mbox{[}offset, offset+size) is assigned in y\mbox{[}\mbox{]};
\item the internal \mbox{\hyperlink{struct_compact_star_1_1_physics_1_1_evolution_1_1_state_layout_1_1_block}{Block}} entry is marked active.
\end{DoxyItemize}

Blocks for tags not listed in {\ttfamily tags} remain inactive.
