\doxysubsection{Detailed Description}
Minimal JSON writer for diagnostics packets. 

We intentionally do not use a full JSON library\+:
\begin{DoxyItemize}
\item avoids bringing in dependencies,
\item keeps output deterministic and easy to diff,
\item sufficient for tooling/regression tests.
\end{DoxyItemize}

Output styles supported\+:
\begin{DoxyItemize}
\item JSON object (single packet)
\item JSONL (one JSON object per line) for append-\/only logs.
\end{DoxyItemize}

Determinism\+:
\begin{DoxyItemize}
\item Scalars iterate in key order due to std\+::map.
\item Text arrays preserve insertion order. 
\end{DoxyItemize}
