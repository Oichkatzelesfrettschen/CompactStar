\doxysubsection{Detailed Description}
Surface photon cooling driver (redshifted frame). 

{\bfseries{Depends on\+:}} \mbox{\hyperlink{namespace_compact_star_1_1_physics_1_1_thermal}{Thermal}} {\bfseries{Updates\+:}} \mbox{\hyperlink{namespace_compact_star_1_1_physics_1_1_thermal}{Thermal}}

This driver\+: 1) obtains (T\+\_\+\textbackslash{}infty) from Thermal\+State, 2) determines a local (T\+\_\+\{\textbackslash{}rm surf\}) using the selected surface/envelope model, 3) computes (L\+\_\+\{\textbackslash{}gamma,\textbackslash{}infty\}) using geometry and redshift factors, 4) accumulates (-\/L\+\_\+\{\textbackslash{}gamma,\textbackslash{}infty\}/(C\+\_\+\{\textbackslash{}rm eff\}T\+\_\+\textbackslash{}infty)) into the thermal RHS.

It does not compute (C\+\_\+\{\textbackslash{}rm eff\}) from EOS/microphysics (yet), and it does not evolve a multi-\/zone temperature profile (isothermal-\/core assumption).
