Adds a photon-\/cooling contribution to the thermal RHS (redshifted frame).

\item Determines a surface temperature (T\+\_\+\{\textbackslash{}rm surf\}) (K) from the \mbox{\hyperlink{namespace_compact_star_1_1_physics_1_1_thermal}{Thermal}} state\+:
\begin{DoxyItemize}

\item via an envelope mapping (future), or
\item via a simple approximation from (T\+\_\+\textbackslash{}infty).
\end{DoxyItemize}
\item Computes (L\+\_\+\{\textbackslash{}gamma,\textbackslash{}infty\}) using geometry + Stefanâ€“\+Boltzmann.
\item Adds (-\/L\+\_\+\{\textbackslash{}gamma,\textbackslash{}infty\}/\+C\+\_\+\{\textbackslash{}rm eff\}) to the RHS for the evolved thermal DOF.

Add photon cooling contribution to the thermal RHS. 

This method must {\bfseries{accumulate}} into {\ttfamily d\+Ydt} (never overwrite).\hypertarget{class_compact_star_1_1_physics_1_1_driver_1_1_thermal_1_1_photon_cooling_autotoc_md16}{}\doxyparagraph{Algorithm outline}\label{class_compact_star_1_1_physics_1_1_driver_1_1_thermal_1_1_photon_cooling_autotoc_md16}

\begin{DoxyEnumerate}
\item Read \mbox{\hyperlink{namespace_compact_star_1_1_physics_1_1_thermal}{Thermal}} state from {\ttfamily Y} and obtain (T\+\_\+\textbackslash{}infty).
\item Determine (T\+\_\+\{\textbackslash{}rm surf\}) according to Options\+::surface\+\_\+model\+:
\begin{DoxyItemize}
\item Direct\+TSurf\+: use Thermal\+State\+::\+T\+\_\+surf if set (\texorpdfstring{$>$}{>}0), else fallback.
\item Envelope\+Mapping\+: if ctx.\+envelope exists, use it (future).
\item Approx\+From\+Tinf\+: set (T\+\_\+\{\textbackslash{}rm surf\} \textbackslash{}leftarrow T\+\_\+\textbackslash{}infty).
\end{DoxyItemize}
\item Determine (A\+\_\+\textbackslash{}infty) from geometry (preferred)\+:  \[
      A_\infty = 4\pi R^2 e^{2\nu(R)} .
   \] taking care to convert (R) into cm if (\textbackslash{}sigma\+\_\+\{\textbackslash{}rm SB\}) is cgs.
\item Form the luminosity and the physical cooling rate\+:  \[
     L_{\gamma,\infty} = \mathrm{global\_scale}\;
                         \mathrm{radiating\_fraction}\;
                         A_\infty\;
                         \sigma_{\rm SB}\;
                         T_{\rm surf}^4 ,
   \]  \[
     \frac{dT_\infty}{dt} = -\frac{L_{\gamma,\infty}}{C_{\rm eff}} .
   \]
\item Convert to the evolved ODE variable and accumulate into the RHS\+:  \[
     \frac{d\ln (T_\infty / T_{\rm ref})}{dt} \mathrel{+}= \frac{1}{T_\infty}\,\frac{dT_\infty}{dt}
     = -\frac{L_{\gamma,\infty}}{C_{\rm eff}\,T_\infty}.
   \]
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Compact\+Star evolves (\textbackslash{}ln (T\+\_\+\textbackslash{}infty / T\+\_\+\{\textbackslash{}rm ref\})) (not (T\+\_\+\textbackslash{}infty)) in the thermal state vector. Drivers must therefore contribute (d(\textbackslash{}ln (T\+\_\+\textbackslash{}infty / T\+\_\+\{\textbackslash{}rm ref\}))/dt).
\end{DoxyNote}
