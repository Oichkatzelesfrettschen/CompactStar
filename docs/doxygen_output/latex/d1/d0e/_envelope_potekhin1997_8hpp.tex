\doxysubsection{Detailed Description}
Potekhin-\/style Tb→\+Ts envelope fits (1997-\/era / classic blanketing envelope models). 

This header declares concrete implementations of the Boundary\+::\+IEnvelope interface using Potekhin-\/style analytic fits. These provide the {\bfseries{thermal boundary condition}} relating the local temperature at the base of the blanketing envelope (Tb) to the local effective surface temperature (Ts).\hypertarget{_envelope_potekhin1997_8hpp_autotoc_md5}{}\doxysubsubsection{Definitions and conventions}\label{_envelope_potekhin1997_8hpp_autotoc_md5}

\begin{DoxyItemize}
\item {\bfseries{Tb}}\+: local temperature at the base of the envelope \mbox{[}K\mbox{]}. In Compact\+Star, Tb is typically evaluated at a density-\/defined boundary (e.\+g., $\rho_b \sim 10^{10}\,\mathrm{g\,cm^{-3}}$), chosen via Boundary\+::\+Tb\+Definition / Boundary\+::\+Find\+Tb\+Index().
\item {\bfseries{Ts}}\+: local effective surface temperature \mbox{[}K\mbox{]}.
\item {\bfseries{g14}}\+: surface gravity in units of $10^{14}\,\mathrm{cm\,s^{-2}}$, typically computed via Boundary\+::\+Surface\+Gravity\+\_\+g14().
\item {\bfseries{xi}}\+: light-\/element column / accreted-\/envelope parameter (dimensionless “knob” used by accreted-\/envelope fits). For pure iron envelopes, xi is ignored.
\end{DoxyItemize}\hypertarget{_envelope_potekhin1997_8hpp_autotoc_md6}{}\doxysubsubsection{Scope}\label{_envelope_potekhin1997_8hpp_autotoc_md6}

\begin{DoxyItemize}

\item No star-\/structure queries occur here; callers supply Tb and g14 explicitly.
