\doxysubsection{Detailed Description}
RAII front-\/end for a GSL ODE driver. 

Wraps stepper choice, tolerances, and integrates the flat ODE vector y\mbox{[}\mbox{]} while delegating the RHS evaluation to {\ttfamily Evolution\+System}.

Design\+:
\begin{DoxyItemize}
\item Dimension N of the system is provided at construction.
\item The integrator does {\itshape not} know about State\+Vector/\+State\+Layout; callers are responsible for packing/unpacking via State\+Packing helpers.
\item The RHS callback simply forwards to {\ttfamily Evolution\+System\+::operator()}. 
\end{DoxyItemize}
