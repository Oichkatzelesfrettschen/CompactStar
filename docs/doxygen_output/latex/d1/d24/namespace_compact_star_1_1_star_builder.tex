\begin{DoxyEnumerate}

\item if we’re at the upper-\/mass end\+: just load that profile
\item else\+: load the neighbor and linearly interpolate
\item compute η\+\_\+I from sequence derivatives
\item locate blanket radius
\item (optional) build DUrca mask
