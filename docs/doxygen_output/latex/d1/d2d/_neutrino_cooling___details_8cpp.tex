\doxysubsection{Detailed Description}
Out-\/of-\/line implementation of Neutrino\+Cooling shared computations. 

Rationale\+:
\begin{DoxyItemize}
\item Keep heavy computations and structure/microphysics dereferences out of headers.
\item Avoid incomplete-\/type problems when using ctx-\/provided objects.
\item Centralize neutrino luminosity computation so RHS and diagnostics agree. 
\end{DoxyItemize}

Compute derived quantities for Neutrino\+Cooling. 

This function should\+: 1) extract Tinf from Thermal\+State, 2) validate driver options (C\+\_\+eff, global\+\_\+scale, enabled channels), 3) compute L\+\_\+nu\+\_\+inf by integrating emissivities over proper volume (future), 4) convert to d\+Tinf/dt = -\/L\+\_\+nu\+\_\+inf / C\+\_\+eff, 5) convert to d/dt ln(Tinf/\+Tref) = (1/\+Tinf) d\+Tinf/dt.
