\doxysubsection{Detailed Description}
A simple runtime contract describing unit conventions and assumptions. 

The contract is intended to be\+:
\begin{DoxyItemize}
\item stable (changes only when conventions/assumptions change),
\item human-\/readable (printed in logs and exported),
\item tool-\/friendly (line-\/based text). 
\end{DoxyItemize}
