\doxysubsection{Detailed Description}
Immutable, per-\/star adapter exposing cached geometry and composition. 

Constructed from an existing {\ttfamily \mbox{\hyperlink{class_compact_star_1_1_n_star}{Compact\+Star\+::\+NStar}}} and (optionally) an EOS object. Provides fast getters for radius grid, metric factors e^{\nu}, e^{\Lambda}, and commonly used columns (density, proton fraction, etc.) by forwarding to Core datasets. Geometry-\/only computations that do not change during evolution can be cached here at construction time (done in \mbox{\hyperlink{class_compact_star_1_1_chemical_heating_1_1_geometry_cache}{Geometry\+Cache}}).
