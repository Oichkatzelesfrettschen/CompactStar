\doxysubsection{Detailed Description}
Composite view over the current ODE state (tag → State$\ast$). 

This class does not store any actual data; it only keeps pointers to State-\/derived objects, indexed by State\+Tag. It is intended to be constructed and owned by the evolution System, then passed by const reference into driver calls.

Ownership\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_state_vector}{State\+Vector}} is {\bfseries{non-\/owning}}; the caller must ensure the underlying \mbox{\hyperlink{namespace_compact_star_1_1_physics_1_1_state}{State}} objects outlive the \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_state_vector}{State\+Vector}}. 
\end{DoxyItemize}
