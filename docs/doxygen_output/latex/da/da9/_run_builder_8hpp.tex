\doxysubsection{Detailed Description}
Small orchestration helpers to reduce boilerplate in Evolution debug mains. 

This module provides narrowly-\/scoped “builder” utilities for\+:
\begin{DoxyItemize}
\item configuring Evolution\+::\+Config with common defaults,
\item wiring Driver\+Context from already-\/built Star\+Context/\+Geometry\+Cache,
\item building State\+Vector, State\+Layout, RHSAccumulator consistently,
\item collecting diagnostics-\/capable drivers.
\end{DoxyItemize}

Important\+: this is NOT intended to be a “simulation framework” that builds stars, chooses EOS, or owns application concerns. Star construction belongs to Core (e.\+g. NStar, Solve\+TOV\+\_\+\+Profile) or higher-\/level application code.
