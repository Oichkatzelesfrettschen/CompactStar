\doxysubsection{Detailed Description}
Right-\/hand side (RHS) functor for the ODE system d\+Y/dt. 

The RHS assembles volume integrals using Geometry\+Cache and Rate\+Set, and computes time derivatives for $T^\infty$ and $\boldsymbol{\eta}$ (and optionally $\Omega$). Designed to be wrapped by a GSL driver in {\ttfamily Integrator\+GSL}.
