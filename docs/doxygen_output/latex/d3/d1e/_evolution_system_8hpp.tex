\doxysubsection{Detailed Description}
Right-\/hand side (RHS) functor for the ODE system d\+Y/dt. 

Evolution\+System wires together\+:
\begin{DoxyItemize}
\item static model context (Star\+Context, Geometry\+Cache, envelope, Config),
\item the dynamic State\+Vector (Spin/\+Thermal/\+Chem/\+BNV blocks),
\item an RHSAccumulator for d\+Y/dt contributions, and
\item a set of physics drivers implementing IDriver.
\end{DoxyItemize}

It does {\bfseries{not}} implement any physics directly. The workflow is\+:


\begin{DoxyEnumerate}
\item Unpack the flat y\mbox{[}\mbox{]} into State blocks (State\+Packing\+::\+Unpack\+State\+Vector).
\item Clear RHSAccumulator.
\item For each driver\+: driver-\/\texorpdfstring{$>$}{>}Accumulate\+RHS(t, state, rhs, runtime\+\_\+context);
\item Scatter RHSAccumulator back into dydt\mbox{[}\mbox{]} (State\+Packing\+::\+Scatter\+RHSFrom\+Accumulator).
\end{DoxyEnumerate}

Designed to be wrapped by a GSL driver.
