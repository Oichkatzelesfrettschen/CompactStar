\doxysubsection{Detailed Description}
Thermal evolution state block for a compact star. 

This class provides the {\bfseries{dynamical thermal degrees of freedom}} used by the evolution system (typically the redshifted internal temperature T∞, or multiple temperature components if needed), as well as optional auxiliary temperatures used by microphysics or cooling models.
