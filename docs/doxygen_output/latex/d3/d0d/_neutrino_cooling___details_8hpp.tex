\doxysubsection{Detailed Description}
Shared computations for Neutrino\+Cooling (physics + diagnostics). 

This module centralizes all computations used by both\+:
\begin{DoxyItemize}
\item Neutrino\+Cooling\+::\+Accumulate\+RHS (evolution physics), and
\item Neutrino\+Cooling\+::\+Diagnose\+Snapshot (diagnostic output).
\end{DoxyItemize}

The key design goal is to prevent \char`\"{}diagnostics drift\char`\"{}\+: if the neutrino cooling model changes, diagnostics remain consistent automatically because they call the same Compute\+Derived() helper.

Compute derived quantities for Neutrino\+Cooling. 

This function should\+: 1) extract Tinf from Thermal\+State, 2) validate driver options (C\+\_\+eff, global\+\_\+scale, enabled channels), 3) compute L\+\_\+nu\+\_\+inf by integrating emissivities over proper volume (future), 4) convert to d\+Tinf/dt = -\/L\+\_\+nu\+\_\+inf / C\+\_\+eff, 5) convert to d/dt ln(Tinf/\+Tref) = (1/\+Tinf) d\+Tinf/dt.
