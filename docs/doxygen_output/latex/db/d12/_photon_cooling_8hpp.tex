Driver that contributes surface photon cooling to the thermal evolution RHS. 

\hypertarget{_photon_cooling_8hpp_autotoc_md5}{}\doxysubsubsection{Physical intent}\label{_photon_cooling_8hpp_autotoc_md5}
This driver adds a {\bfseries{photon luminosity loss}} term to the redshifted thermal degree of freedom (typically (T\+\_\+\textbackslash{}infty)), producing a contribution

\[
  \frac{d\ln (T_\infty / T_{\rm ref})}{dt} = - \frac{L_{\gamma,\infty}}{C_{\rm eff}\,T_\infty}
\]

\begin{DoxyNote}{Note}
{\bfseries{ODE variable convention\+:}} Compact\+Star evolves the dimensionless logarithmic variable (\textbackslash{}ln (T\+\_\+\textbackslash{}infty / T\+\_\+\{\textbackslash{}rm ref\})). If a physical model naturally computes (d\+T\+\_\+\textbackslash{}infty/dt), it must be converted via\+:  \[
  \frac{d\ln (T_\infty / T_{\rm ref})}{dt} = \frac{1}{T_\infty}\,\frac{dT_\infty}{dt}.

\end{DoxyNote}

\[
  L_{\gamma,\infty} = \mathcal{F}_{\rm rad}\; A_{\infty}\; \sigma_{\rm SB}\; T_{\rm surf}^4 .

\item $A_{\infty}$ is the {\bfseries{redshifted emitting area}} appropriate for luminosity measured at infinity.
\item $\mathcal{F}_{\rm rad}$ is a {\bfseries{dimensionless}} prefactor capturing partial surface emission (hot spots, beaming, opacity fudge, etc.).
