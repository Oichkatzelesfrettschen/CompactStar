\doxysubsection{Detailed Description}
Mapping between flat ODE vector indices and State\+Tag blocks. 

The evolution system uses a single flat ODE vector y\mbox{[}\mbox{]} (and dydt\mbox{[}\mbox{]}) that concatenates all active state blocks (Spin\+State, Thermal\+State, Chem\+State, BNVState, ...). State\+Layout records\+:


\begin{DoxyItemize}
\item which State\+Tag blocks are active,
\item where each block starts in y\mbox{[}\mbox{]} (offset),
\item how many components each block has (size),
\item the total dimension of the system.
\end{DoxyItemize}

Typical usage\+: 1) Build and register State blocks in a State\+Vector. 2) Call State\+Layout\+::\+Configure(state, \{State\+Tag\+::\+Thermal, State\+Tag\+::\+Chem, ...\}). 3) Use Offset(tag) and Block\+Size(tag) when packing/unpacking y\mbox{[}\mbox{]}.

This keeps the \char`\"{}wiring\char`\"{} between tagged State blocks and their positions in the global ODE vector explicit and easy to audit.
