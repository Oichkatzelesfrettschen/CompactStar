\doxysubsection{Detailed Description}
Potekhin-\/style light-\/element (accreted) envelope fit. 

Returns $T_s(T_b, g_{14}, \xi)$ for accreted/light-\/element envelopes. The parameter {\ttfamily xi} controls the amount of light elements (model-\/dependent).

\begin{DoxyNote}{Note}
Input validation (e.\+g., Tb\texorpdfstring{$>$}{>}0, g14\texorpdfstring{$>$}{>}0, finite xi) is performed in the .cpp implementation. 
\end{DoxyNote}
