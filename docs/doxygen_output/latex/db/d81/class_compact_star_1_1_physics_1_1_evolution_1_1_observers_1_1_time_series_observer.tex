\doxysubsection{Detailed Description}
Minimal time-\/series recorder for selected scalar quantities. 

The \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_observers_1_1_time_series_observer}{Time\+Series\+Observer}} writes a single table-\/like stream where each row corresponds to a recorded sample time.

It is meant to produce a compact dataset suitable for\+:
\begin{DoxyItemize}
\item quick plotting (Python, MATLAB, gnuplot),
\item quick inspection,
\item numerical regression comparisons at the level of {\itshape state variables}.
\end{DoxyItemize}

It is {\itshape not} meant to replace \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_observers_1_1_diagnostics_observer}{Diagnostics\+Observer}} for deep debugging, driver contracts, or per-\/driver introspection.
