Solve the TOV equations for a {\itshape single neutron star} specified by a target gravitational mass, returning the full radial structure as a vector of \mbox{\hyperlink{struct_compact_star_1_1_core_1_1_t_o_v_point}{TOVPoint}}. 

This routine performs an internal search over central energy density ( \textbackslash{}varepsilon\+\_\+c ) to find the value that yields the requested gravitational mass (in solar masses). The EOS must have been imported beforehand via \mbox{\hyperlink{class_compact_star_1_1_core_1_1_t_o_v_solver_a7aceb2cc760d1fd93821631a9f7d8989}{Import\+EOS}}.\hypertarget{class_compact_star_1_1_core_1_1_t_o_v_solver_autotoc_md3}{}\doxyparagraph{Algorithm summary}\label{class_compact_star_1_1_core_1_1_t_o_v_solver_autotoc_md3}

\begin{DoxyEnumerate}
\item Construct a coarse, logarithmically spaced grid in ( \textbackslash{}varepsilon\+\_\+c ) between the allowed EOS range (with the usual floor/ceiling safety margins).
\item For each sampled ( \textbackslash{}varepsilon\+\_\+c )\+:
\begin{DoxyItemize}
\item Integrate the TOV equations using \mbox{\hyperlink{class_compact_star_1_1_core_1_1_t_o_v_solver_a70729eb1f9707377347f2f01b7006f99}{Single\+Star\+Solve\+To\+TOVPoints}},
\item Record the resulting mass ( M(\textbackslash{}varepsilon\+\_\+c) ).
\end{DoxyItemize}
\item Identify a {\itshape stable branch} interval where $ M(\varepsilon_{c,i+1}) > M(\varepsilon_{c,i}) $ and where ( M(\textbackslash{}varepsilon\+\_\+c) ) brackets the target mass.
\item If such a bracket exists, perform a bisection search in ( \textbackslash{}varepsilon\+\_\+c ) on that monotonic segment until a desired mass tolerance is achieved.
\item If a stable-\/branch bracketing interval cannot be found (e.\+g. EOS with no monotonic segment covering the target), fall back to returning the profile whose mass is closest among the coarse samples.
\end{DoxyEnumerate}\hypertarget{class_compact_star_1_1_core_1_1_t_o_v_solver_autotoc_md4}{}\doxyparagraph{Output}\label{class_compact_star_1_1_core_1_1_t_o_v_solver_autotoc_md4}

\begin{DoxyItemize}
\item On {\bfseries{success}}, {\ttfamily out\+\_\+tov} is filled with the radial grid $ (r, m, \nu\textnormal{\textquotesingle}, \nu, p, \epsilon, n_B, \rho_i) $ produced by TOV integration, and the return value is the number of radial points.
\item If {\ttfamily out\+\_\+species\+\_\+labels} is non-\/null, it is filled with the species labels inferred from the EOS (typically EOSTable\+::extra\+\_\+labels).
\item On {\bfseries{failure}}, the function returns {\ttfamily 0} and leaves {\ttfamily out\+\_\+tov} empty.
\end{DoxyItemize}
