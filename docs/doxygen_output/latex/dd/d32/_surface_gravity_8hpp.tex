\doxysubsection{Detailed Description}
Helpers to compute neutron-\/star surface gravity in the \char`\"{}g14\char`\"{} convention. 

The envelope literature typically parameterizes the surface gravity as\+:

g14 ≡ g / (1e14 cm s$^\wedge$-\/2)

and then provides Ts(\+Tb, g14, ...).

In GR, a commonly used effective surface gravity is\+:

g\+\_\+s = (G M / R$^\wedge$2) $\ast$ (1 -\/ 2GM/(Rc$^\wedge$2))$^\wedge$\{-\/1/2\}

(equivalently g\+\_\+s = GM e$^\wedge$\{-\/ν(\+R)\} / R$^\wedge$2 with e$^\wedge$\{ν(\+R)\} = sqrt(1-\/2GM/\+Rc$^\wedge$2) for a Schwarzschild exterior).

Compact\+Star stores structure in geometric units internally; this helper avoids forcing any particular unit system into the envelope code. It computes g14 from Star\+Context/\+Geometry\+Cache using the metric and/or compactness.

Requirements\+:
\begin{DoxyItemize}
\item Star\+Context provides surface radius R (km) and gravitational mass M (Msun or km), OR provides exterior metric factor at the surface (e$^\wedge$\{ν(\+R)\} or e$^\wedge$\{2ν(\+R)\}).
\item If Geometry\+Cache is available, it may provide R and exp(2ν) at the surface.
\end{DoxyItemize}

If your Star\+Context API differs, adapt the small “adapter” functions in the .hpp.

Surface gravity in g14 units\+: g14 ≡ g / (1e14 cm s$^\wedge$-\/2). 

Return g14 = g / (1e14 cm s$^\wedge$-\/2).

Assumptions / conventions\+:
\begin{DoxyItemize}
\item R\+\_\+km is the circumferential radius at the surface \mbox{[}km\mbox{]}.
\item M\+\_\+km is the gravitational mass in geometric length units \mbox{[}km\mbox{]}, i.\+e. M\+\_\+km = GM/c$^\wedge$2 expressed in km.
\item expnu is exp(ν(\+R)), dimensionless. For an exterior Schwarzschild metric expnu = sqrt(1 -\/ 2 M\+\_\+km / R\+\_\+km).
\end{DoxyItemize}

Then\+: g\+\_\+cgs = c$^\wedge$2 $\ast$ M\+\_\+cm / (R\+\_\+cm$^\wedge$2 $\ast$ expnu)

which matches\+: g = GM/\+R$^\wedge$2 / sqrt(1 -\/ 2GM/(Rc$^\wedge$2)).

This tries (in order)\+: 1) Use Geometry\+Cache if it exposes R and exp(2ν) at the surface. 2) Use Star\+Context profile view if it exposes R and ν(\+R) / exp(ν(\+R)). 3) Fallback\+: if only compactness C = 2GM/(Rc$^\wedge$2) is available, use g\+\_\+s = GM/\+R$^\wedge$2 / sqrt(1 -\/ C).
