\doxysubsection{Detailed Description}
\mbox{\hyperlink{struct_compact_star_1_1_physics_1_1_evolution_1_1_observers_1_1_time_series_observer_1_1_column}{Column}} specification for \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_evolution_1_1_observers_1_1_time_series_observer}{Time\+Series\+Observer}} output. 

A column is defined by\+:
\begin{DoxyItemize}
\item a stable column name (header label),
\item how to obtain the value at each sample,
\item optional provenance metadata (not repeated per row).
\end{DoxyItemize}

Notes\+:
\begin{DoxyItemize}
\item {\ttfamily key} should be tooling-\/friendly and stable (snake\+\_\+case recommended).
\item {\ttfamily unit} and {\ttfamily description} are written once (optional) via a sidecar file if enabled; they are not repeated in the table itself.
\item For Column\+Source\+::\+Driver\+Scalar, {\ttfamily catalog\+\_\+ref} must be set. 
\end{DoxyItemize}
