\doxysubsection{Detailed Description}
Enumeration of physical state components used by evolution drivers. 

Provides canonical identifiers ({\ttfamily State\+Tag}) for each sub-\/state (spin, thermal, chemical, BNV, etc.) so that drivers and the evolution graph can declare dependencies and update targets in a type-\/safe manner.
