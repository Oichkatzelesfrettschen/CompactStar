\doxysubsection{Detailed Description}
Observer that periodically records driver diagnostics to JSONL. 

This observer is intended to be\+:
\begin{DoxyItemize}
\item separate from physics (no effect on integration),
\item deterministic output for regression testing,
\item flexible about what drivers expose diagnostics (opt-\/in interface).
\end{DoxyItemize}

Operation\+:
\begin{DoxyItemize}
\item At Observe(t, Y, ctx) it checks whether it\textquotesingle{}s time to record.
\item It queries each registered driver diagnostics interface and writes one JSON object per driver (JSONL) into an output file.
\end{DoxyItemize}

Output format\+:
\begin{DoxyItemize}
\item JSONL (one object per line), which scales and is tooling-\/friendly.
\item Each line includes producer/time/step/scalars/contract/messages. 
\end{DoxyItemize}
