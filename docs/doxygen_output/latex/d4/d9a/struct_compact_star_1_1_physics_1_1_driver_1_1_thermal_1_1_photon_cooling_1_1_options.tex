\doxysubsection{Detailed Description}
Configuration parameters for \mbox{\hyperlink{class_compact_star_1_1_physics_1_1_driver_1_1_thermal_1_1_photon_cooling}{Photon\+Cooling}}. 

Design goals\+:
\begin{DoxyItemize}
\item Parameters should be physically interpretable.
\item All scale factors are dimensionless.
\item Temperatures are in Kelvin.
\item Heat capacity is in erg/K (if using cgs σ\+\_\+\+SB).
\end{DoxyItemize}

Validation policy (recommended in implementation)\+:
\begin{DoxyItemize}
\item If radiating\+\_\+fraction \texorpdfstring{$<$}{<}= 0 or global\+\_\+scale \texorpdfstring{$<$}{<}= 0\+: treat as intentionally disabled and produce zero RHS contribution (no exception).
\item If C\+\_\+eff \texorpdfstring{$<$}{<}= 0\+: this is physically invalid; throw or log+skip (choose one policy). 
\end{DoxyItemize}
