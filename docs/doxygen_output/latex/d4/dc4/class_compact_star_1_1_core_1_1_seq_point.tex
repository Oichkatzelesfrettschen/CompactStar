\doxysubsection{Detailed Description}
Holds a single data point in a star sequence. 

\hypertarget{class_compact_star_1_1_core_1_1_seq_point_autotoc_md2}{}\doxyparagraph{Stored quantities (double)\+:}\label{class_compact_star_1_1_core_1_1_seq_point_autotoc_md2}

\begin{DoxyItemize}
\item {\ttfamily ec} \+: central energy density (code units; match the solver)
\item {\ttfamily m} \+: gravitational mass at the surface (M\+\_\+\textbackslash{}odot)
\item {\ttfamily r} \+: circumferential radius at the surface (km)
\item {\ttfamily pc} \+: central pressure (same unit system as the solver output)
\item {\ttfamily b} \+: baryon number integral (dimensionless or code units, per our convention)
\item {\ttfamily I} \+: moment of inertia (e.\+g., g·cm² or km³, depending on internal units)
\end{DoxyItemize}

This class is intentionally minimal and trivially copyable so it can be passed by value in vectors/containers and written to text rows easily.
