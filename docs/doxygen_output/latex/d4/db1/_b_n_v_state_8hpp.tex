\doxysubsection{Detailed Description}
Dynamic or cached quantities associated with baryon-\/number violation. 

This state block provides storage for BNV-\/related diagnostics or dynamic variables that may appear in BNV evolution drivers (e.\+g. ψ-\/population evolution, effective η-\/like variables, or observationally relevant meta-\/parameters).

Depending on your BNV model, this may store\+:
\begin{DoxyItemize}
\item a small vector of ODE-\/evolved quantities, or
\item a cached set of derived diagnostic values (non-\/\+ODE), or both.
\end{DoxyItemize}

Present version implements a simple ODE-\/compatible state with two components\+:
\begin{DoxyEnumerate}
\item η\+\_\+I — dimensionless imbalance parameter (model-\/specific)
\item spin\+\_\+down\+\_\+limit — Γ\+\_\+\+BNV limit inferred from pulsar spin-\/down
\end{DoxyEnumerate}

Additional DOFs can be added without touching the evolution core.
