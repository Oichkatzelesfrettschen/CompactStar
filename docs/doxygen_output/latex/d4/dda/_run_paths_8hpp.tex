\doxysubsection{Detailed Description}
Standardized run output paths for Evolution-\/based runs. 

This header defines a small, deterministic convention for\+:
\begin{DoxyItemize}
\item log file path
\item diagnostics JSONL path
\item diagnostics catalog JSON path
\item time-\/series CSV path
\item time-\/series sidecar metadata path
\end{DoxyItemize}

Motivation\+:
\begin{DoxyItemize}
\item Debug mains and examples tend to duplicate output path wiring.
\item Observers expect consistent conventions (catalog â†’ timeseries, etc.).
\item Keeping this in Evolution (not in Observers) centralizes run I/O policy.
\end{DoxyItemize}

Design principles\+:
\begin{DoxyItemize}
\item Pure data container + deterministic builder.
\item No side effects (does not create directories).
\item No dependency on core star-\/building machinery. 
\end{DoxyItemize}
