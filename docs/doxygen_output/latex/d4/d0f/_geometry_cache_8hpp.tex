\doxysubsection{Detailed Description}
Precomputed geometric weights and masks for fast radial integration. 

Geometry\+Cache builds redshift/volume weights once per star\+:
\begin{DoxyItemize}
\item $w_C(r)=4\pi r^2 e^{\Lambda(r)}$ (heat capacity integrals)
\item $w_\nu(r)=4\pi r^2 e^{\Lambda(r)+2\nu(r)}$ (luminosities at infinity) 
\end{DoxyItemize}
