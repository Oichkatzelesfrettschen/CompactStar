\doxysubsection{Detailed Description}
Precomputed geometric factors for fast radial integration (Data\+Column-\/native). 

Metric convention assumed\+:  \[
  ds^2 = -e^{2\nu(r)} dt^2 + e^{2\Lambda(r)} dr^2 + r^2 d\Omega^2
\]

where\+:
\begin{DoxyItemize}
\item $\nu(r)$ is the time metric exponent
\item $\Lambda(r)$ is the radial metric exponent
\end{DoxyItemize}

This cache stores primitive, reusable columns\+:
\begin{DoxyItemize}
\item $r$ (km)
\item $A(r)=4\pi r^2$
\item $e^{\nu}$, $e^{-\nu}$, $e^{2\nu}$
\item $e^{\Lambda}$, $e^{-\Lambda}$
\item $e^{\nu-\Lambda}$, $e^{-(\nu+\Lambda)}$
\item $w_V(r)=4\pi r^2 e^{\Lambda}$ (proper-\/volume shell weight)
\item $w_V e^{\nu}$, $w_V e^{2\nu}$ (common redshifted variants) 
\end{DoxyItemize}
