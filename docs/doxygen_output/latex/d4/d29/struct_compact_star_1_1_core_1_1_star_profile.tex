\doxysubsection{Detailed Description}
Structural / metric / composition data for a compact-\/star configuration. 

One instance = one star.

It contains\+:
\begin{DoxyItemize}
\item full radial data as a Data\+Set;
\item sequence metadata (M, R, central values) via \mbox{\hyperlink{class_compact_star_1_1_core_1_1_seq_point}{Seq\+Point}};
\item explicit indices for all “core” columns (r, m, ν′, p, ε, ρ, ν, λ);
\item a typed {\ttfamily enum class Column} so new code can use enum and old code can use ints;
\item per-\/species density columns, with names and indices. 
\end{DoxyItemize}
