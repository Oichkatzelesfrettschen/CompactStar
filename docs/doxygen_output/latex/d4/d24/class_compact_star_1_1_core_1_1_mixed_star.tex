\doxysubsection{Detailed Description}
Represents a star composed of both visible and dark matter. 

Encapsulates the core and mantle regions of an admixed star, stores TOV solutions, supports data analysis (e.\+g., moment of inertia), and exports computed profiles.

The class is designed to work with the \mbox{\hyperlink{class_compact_star_1_1_core_1_1_t_o_v_solver}{TOVSolver}} and \mbox{\hyperlink{class_compact_star_1_1_core_1_1_rotation_solver}{Rotation\+Solver}} classes. It provides methods to initialize, append TOV points, and compute various properties such as mass, baryon number, and moment of inertia. It also allows for the evaluation of the metric function and its derivative. The class is capable of exporting the mixed star profile to a file.
