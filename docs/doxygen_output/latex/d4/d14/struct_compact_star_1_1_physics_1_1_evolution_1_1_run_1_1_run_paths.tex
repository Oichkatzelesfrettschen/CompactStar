\doxysubsection{Detailed Description}
Canonical file layout for an Evolution run. 

All members are absolute or run-\/root-\/relative paths depending on how you pass the base directory. No filesystem mutations are performed by this struct.

Typical layout\+: $<$base\+\_\+results\+\_\+dir$>$/$<$out\+\_\+dir$>$/ diagnostics.\+jsonl diagnostics.\+catalog.\+json timeseries.\+csv timeseries.\+csv.\+meta.\+json $<$optional logs$>$
