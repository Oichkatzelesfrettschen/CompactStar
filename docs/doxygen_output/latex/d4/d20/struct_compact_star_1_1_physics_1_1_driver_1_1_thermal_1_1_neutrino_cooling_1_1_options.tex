\doxysubsection{Detailed Description}
Configuration knobs controlling which neutrino channels are enabled. 

These options are intended to map to future microphysics channel selection. {\ttfamily global\+\_\+scale} provides a convenient dimensionless knob for debugging/tests.

\begin{DoxyNote}{Note}
For a physically meaningful model, you will eventually also want a policy for (C\+\_\+\{\textbackslash{}rm eff\}) (either in \mbox{\hyperlink{struct_compact_star_1_1_physics_1_1_driver_1_1_thermal_1_1_neutrino_cooling_1_1_options}{Options}} or supplied by {\ttfamily ctx}). 
\end{DoxyNote}
