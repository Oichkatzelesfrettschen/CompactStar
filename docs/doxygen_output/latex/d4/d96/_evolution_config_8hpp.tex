\doxysubsection{Detailed Description}
User-\/configurable options for chemical/thermal/spin evolution runs. 

Includes integrator tolerances, enabled physics channels, envelope and gap choices, output cadence, and initial-\/condition sizes (e.\+g. n\+\_\+eta).
